Welcome to Rhodeus Script.

Rhodeus Script is a new web based scripting language to create web sites. It is written in D programming language. With this project it is aimed to create a language that is fully object-oriented and programmer-friendly. In some ways PHP is so weak. There is no methods for objects and some unnecessary things like typing semicolon end to the line is optional in Rhodeus Script. Also it is influenced D, Python, Ruby languages. So there is some features from these languages. For example typing * to collect whole parameters in functions is influenced from Python. Here is a sample Rhodeus Script code:

Length of "Hello World" is <|
	echo "Hello World".length
|>.

It is tested for Windows 8 and it works fine. It couldn't work with other operating systems. As soon as We will fix this.

Speed Comparison

We've made some speed tests between other scripting languages and results gives hope to make it better. To creating 1 million array that has 40 elements duration is faster than PHP 6 times.
